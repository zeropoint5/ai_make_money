# AI 创业公司信息集锦

## 项目简介

本项目旨在收集和整理人工智能(AI)领域的创业公司信息，比如 Y Combinator 最近孵化的 AI 创业企业。通过这个项目，我们希望为对AI创业生态感兴趣的研究者、投资者和创业者提供一个便捷的信息参考。

## 内容概览

- Y Combinator最近的 AI 创业公司信息
- 以Markdown表格形式呈现，便于阅读和检索

4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启一个 Pull Request
